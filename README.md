# Tomb of Infernadax

A browser-based dungeon crawler inspired by Wizardry, built from The Five Seals D&D campaign.

Lead a party of seven adventurers through five floors of the Tomb of Infernadax. 
Set commands for your whole party before each combat round resolves — then watch 
the chaos unfold. Find the phylactery. Destroy it. Then kill the dragon.

**Play it:** [tomb-of-infernadax-production.up.railway.app](https://tomb-of-infernadax-production.up.railway.app)

## Stack
- Plain HTML/CSS/JS frontend (no frameworks)
- Node.js + Express backend
- Hosted on Railway, auto-deployed from this repo

## Development
See `DESIGN.md` for the full game design document.
```bash
npm install
node server.js
```
